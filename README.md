# LavLossCalculation Application

This is a simple web-based calculator application called **LavLossCalculation**. It allows users to add multiple rows to two columns, input numbers, and calculate the result. The result shows the difference between the total of **Positive Value** (Column 1) and **Negative Value** (Column 2). The table rows also change color based on the values.
